# Roulette

シンプルなブラウザ用ルーレットページです。`index.html` を開くだけで動作し、キャンバス描画のホイールがスピンします。

## 特徴
- 参加者を改行区切りで入力し、スピンで当選者を表示
- 名前のシャッフル、コピー／貼り付けボタン付き
- 針は上部中央で停止位置を示し、キャンバスにシンプルなグラデーションと軽い揺れアニメーション

## 使い方
1. このリポジトリを clone するか、`index.html` をローカルに保存
2. ブラウザで `index.html` を開く
3. テキストエリアに名前を入力（改行区切り）
4. 必要なら「ランダム順にする」「コピー」「貼り付け」を使用
5. 「スピンする」でホイールを回し、上部の針の位置にあるセグメントが当選

## 開発メモ
- 依存パッケージなしの純粋な HTML/CSS/JS
- スタイルはページ内の `<style>` とスクリプトは `<script>` に内包
